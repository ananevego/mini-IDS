# Mini-IDS
Mini-IDS in python for traning, using scapy

## Overview
Featutes:
- Captures TCP and UDP packets
- Detects multiple connections from the same IP adress
- Detects high levels of DNS traffic
- Detects connections to unusual ports
- Outupt alerts on the console
- Logs all traffic to traffic.log

## Launch
1. Install dependencies
   ```bash
   pip install -r requirements.txt
2. Run
   ```bash
   sudo python main.py
